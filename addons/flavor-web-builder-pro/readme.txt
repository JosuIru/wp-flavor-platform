Constructor visual drag & drop de páginas y landing pages con IA integrada para Flavor Platform.

== Description ==

**Flavor Web Builder Pro** es un constructor visual avanzado que te permite crear páginas y landing pages profesionales sin necesidad de código.

= Características Principales =

* **Constructor Drag & Drop**: Arrastra y suelta componentes para construir tus páginas
* **170+ Componentes**: Héroes, servicios, testimonios, formularios, galerías y más
* **17 Templates Predefinidos**: Plantillas listas para restaurante, hotel, clínica, gimnasio, etc.
* **Asistente IA**: Genera diseños completos automáticamente con inteligencia artificial
* **Vista Previa en Tiempo Real**: Ve los cambios al instante mientras editas
* **Responsive**: Todos los componentes se adaptan a móviles y tablets
* **Tailwind CSS**: Utiliza el framework de diseño más moderno
* **Custom Post Type**: Gestiona landing pages independientes de páginas normales

= Integración con IA =

Si tienes configurado un motor de IA en Flavor Platform (Claude, OpenAI, etc.), puedes usar el **Asistente IA** para:

* Generar landing pages completas automáticamente
* Sugerir componentes según tu sector
* Crear contenido optimizado
* Adaptar diseños existentes

== Installation ==

1. Asegúrate de tener **Flavor Platform 3.0** instalado y activado
2. Sube la carpeta `flavor-web-builder-pro` a `/wp-content/plugins/`
3. Activa el plugin desde el menú 'Plugins' en WordPress
4. El addon se activará automáticamente en Flavor Platform > Addons
5. Ve a "Landing Pages" en el menú lateral para crear tu primera página

== Frequently Asked Questions ==

= ¿Necesito Flavor Platform para usar este addon? =

Sí, este addon requiere que Flavor Platform 3.0 o superior esté instalado y activado.

= ¿Puedo usar el Web Builder sin IA? =

Sí, el constructor funciona perfectamente sin IA. El asistente IA es una característica opcional adicional.

= ¿Los diseños son responsive? =

Sí, todos los componentes y templates están optimizados para móviles, tablets y desktop.

= ¿Puedo editar el código HTML/CSS? =

El Web Builder está diseñado para no necesitar código, pero puedes agregar CSS personalizado en cada componente.

= ¿Cuántas landing pages puedo crear? =

Ilimitadas. No hay restricciones en el número de páginas que puedes crear.

= ¿Funciona con mi tema de WordPress? =

Sí, las landing pages utilizan su propio template y funcionan independientemente de tu tema.
