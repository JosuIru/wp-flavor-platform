# Modelos 3D de Ejemplo

Modelos gratuitos para probar el Editor 3D de VBP.

## Modelos incluidos

| Archivo | Tamaño | Descripción | Licencia |
|---------|--------|-------------|----------|
| `caja.glb` | 2 KB | Caja simple | Khronos (CC0) |
| `cesta-frutas.glb` | 57 KB | Cesta con frutas | CC0 |
| `flamingo.glb` | 76 KB | Flamenco animado | Three.js (MIT) |
| `caballo.glb` | 178 KB | Caballo animado | Three.js (MIT) |
| `persona.glb` | 2.1 MB | Soldado/persona | Three.js (MIT) |

## Cómo usar

1. En VBP Editor, añadir bloque **3D Scene**
2. Dentro, añadir bloque **3D Model**
3. Seleccionar uno de estos archivos
4. Ajustar escala y posición

## Ruta de acceso

```
/wp-content/plugins/flavor-platform/assets/vbp/models/examples/[archivo].glb
```
