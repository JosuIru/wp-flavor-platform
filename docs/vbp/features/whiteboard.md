# Whiteboard (Pizarra)

Modo pizarra infinita para brainstorming, diagramas y planificación visual.

## Qué es

Similar a FigJam o Miro, permite:
- Dibujo libre
- Notas adhesivas
- Conectores y flechas
- Formas básicas
- Colaboración en tiempo real

## Cómo acceder

1. En el editor, clic en icono de pizarra (toolbar superior)
2. O atajo: `Ctrl+Shift+W`

## Herramientas

### Dibujo
| Herramienta | Atajo | Descripción |
|-------------|-------|-------------|
| Lápiz | `P` | Dibujo libre |
| Marcador | `M` | Trazo grueso |
| Borrador | `E` | Borrar trazos |

### Formas
| Forma | Atajo | Descripción |
|-------|-------|-------------|
| Rectángulo | `R` | Rectángulo/cuadrado |
| Círculo | `O` | Círculo/elipse |
| Línea | `L` | Línea recta |
| Flecha | `A` | Flecha direccional |
| Conector | `C` | Línea que conecta elementos |

### Contenido
| Elemento | Atajo | Descripción |
|----------|-------|-------------|
| Nota adhesiva | `S` | Post-it de colores |
| Texto | `T` | Texto libre |
| Imagen | `I` | Insertar imagen |
| Frame | `F` | Marco/contenedor |

## Notas adhesivas

Colores disponibles:
- 🟡 Amarillo (ideas)
- 🟢 Verde (aprobado)
- 🔴 Rojo (problema)
- 🔵 Azul (información)
- 🟣 Morado (pregunta)
- 🟠 Naranja (pendiente)

## Conectores

Los conectores se anclan a elementos y se actualizan al mover:

```
[Elemento A] -----> [Elemento B]
      │
      └──────> [Elemento C]
```

Tipos de línea:
- Recta
- Curva
- Escalonada (ángulos rectos)

## Colaboración

En modo whiteboard, la colaboración muestra:
- Cursores de otros usuarios
- Nombre junto al cursor
- Selecciones en tiempo real
- Historial de cambios

## Exportar

| Formato | Descripción |
|---------|-------------|
| PNG | Imagen del tablero |
| SVG | Vectorial editable |
| PDF | Documento imprimible |
| JSON | Estructura para backup |

## Casos de uso

- **Brainstorming**: Lluvia de ideas con notas
- **User flows**: Diagramas de flujo de usuario
- **Wireframes**: Bocetos rápidos
- **Retrospectivas**: Organizar feedback del equipo
- **Mapas mentales**: Organizar conceptos
- **Planificación**: Roadmaps visuales

## Atajos de teclado

| Atajo | Acción |
|-------|--------|
| `Space+Drag` | Mover canvas |
| `Scroll` | Zoom |
| `Ctrl+0` | Zoom 100% |
| `Ctrl+1` | Zoom a ajustar |
| `Delete` | Eliminar selección |
| `Ctrl+D` | Duplicar |
| `Ctrl+G` | Agrupar |
