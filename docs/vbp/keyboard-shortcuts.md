## Personalizar Atajos

Los atajos se definen en `vbp-keyboard-shortcuts.json`.

### Estructura

```json
{
    "version": "2.0.0",
    "categories": {
        "file": {
            "label": "Archivo",
            "shortcuts": {
                "ctrl+s": "save",
                "ctrl+shift+s": "saveAs"
            }
        }
    }
}
```

### Agregar Atajo Personalizado

```javascript
// Via JavaScript
VBPKeyboard.registerShortcut('ctrl+alt+x', 'miAccion', function() {
    console.log('Atajo ejecutado');
});
```

---

## Notas

- Los atajos funcionan cuando el editor tiene el foco
- Algunos atajos se desactivan durante edicion de texto
- En campos de texto, los atajos de edicion tienen prioridad
- La paleta de comandos (`Ctrl+K`) muestra todos los atajos
