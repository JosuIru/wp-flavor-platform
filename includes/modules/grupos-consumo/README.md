# Módulo Grupos de Consumo

Gestión de pedidos colectivos a productores locales para Flavor Chat IA.

## Características

- Pedidos colectivos con mínimos y máximos
- Participación de usuarios
- Seguimiento de pagos y recogidas
- Integración con chat IA
- Shortcodes para frontend
- API REST para apps móviles
